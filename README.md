# phc-world-board-answer-service
* 마이크로 서비스 아키텍처에서 게시글의 답변 등록, 수정, 조회, 삭제를 담당하고 있는 마이크로 서비스입니다.
***
### Stack
> * JAVA 17
> * Spring-Boot 3.2.1
> * Gradle 8.5
> * Spring-Security
> * JPA, QueryDSL
> * Spring Cloud Eureka Client
> * Spring Cloud Bus
> * RabbitMQ
> * JWT
> * Kafka
> * Kafka Connect
> * MySQL
> * H2-database
> * Swagger
> * JUnit5
